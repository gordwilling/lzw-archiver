Detailed Description

Helper for the Button class. This class handles the serialization of LZW symbol encodings to a stream. In particular, it handles the conversion of variable-length codewords into fixed-width bytes suitable for output to a standard stream. This class begins by using the minimum code word size of 9 bits, gradually increasing that size as needed to accomodate increasing code values.
